# Посмотри в окно

Проектная работа по вёрстке интерфейса приложения «Посмотри в окно» с использованием CSS.  
Цель — стилизовать готовую HTML-разметку и JavaScript-логику в соответствии с предоставленным макетом, не изменяя структуру HTML и сохраняя существующую функциональность.

## Основные задачи
- Сборка лейаута с использованием Flexbox и Grid.
- Точная настройка размеров и позиционирования элементов.
- Стилизация текстов, форм, кнопок и карточек.
- Реализация состояний интерактивных элементов (`:hover`, `:focus-visible`, `:active`).
- Позиционирование прелоадеров над видео и списком карточек.
- Работа с шаблонами (`<template>`) и кастомными элементами форм.

## Особенности
- Используются логические свойства и абсолютные единицы измерения.
- Все интерактивные состояния управляются через CSS.
- Макет фиксированного размера; адаптивность не требуется.
- Запрещено изменять имена классов и структуру HTML.
